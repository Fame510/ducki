Connections

LLM Provider
Stored only in your browser (localStorage). Sent directly to the provider you pick. On-device brains need no key at all — the model downloads once and runs fully on your device (Genius needs Chrome/WebGPU; the light model runs anywhere). Tap Save to load it.
GitHub
Create a token at github.com/settings/tokens (fine-grained, with Repository + Issues access).
Firecrawl
Web scraping/crawling. Get a key at firecrawl.dev. Optional now — web search & page reading work with no key. A Firecrawl key adds premium scraping, structured extraction, and human-like browser automation.
Connections vault
Store an API key for any service with a REST API. DUCKi reaches it through its universal connector — reference a saved key in a request as {{vault:NAME}}; the secret is injected on your device and never sent to the model. This is how DUCKi connects to thousands of services.
Gmail
Create an OAuth Client ID (type: Web), add as an authorized JavaScript origin, and enable the Gmail API. Google Cloud Console ↗
Off by default. The agent can only read until you enable this.
